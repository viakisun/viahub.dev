🎯 Layout Principles

✦ GitHub-Inspired Structure
개발자들에게 친숙한 레이아웃 패턴을 따라 학습 곡선을 최소화
✦ Content-First Design
핵심 콘텐츠가 중앙에 위치하고 부가 정보는 사이드바에 배치
✦ Progressive Disclosure
복잡한 농업 로봇 데이터를 단계적으로 공개하여 인지 부하 감소
✦ Mobile-First Responsive
농장 현장에서의 모바일 사용을 고려한 반응형 설계
✦ Consistent Navigation
모든 페이지에서 일관된 네비게이션 구조와 상태 표시
페이지 3/4 - Layout & Grid System
